# # # 🫀 ECG Signal Processing & Deep Learning Pipeline

This project focuses on processing and analyzing ECG (Electrocardiogram) signals using Python and deep learning techniques, including U-Net architecture.

## 🎯 Key Features

- ECG signal preprocessing  
- Feature extraction  
- Deep learning model training (U-Net)  
- Prediction and analysis pipeline  

## 🧠 Technologies Used
- Python
- NumPy, Pandas
- Jupyter Notebook
- Deep Learning (U-Net architecture)

## 📁 Project Structure
- `data/` → raw and processed datasets
- `models/` → trained models
- `outputs/` → results and predictions
- `*.ipynb` → analysis notebooks
- `train_unet.py` → model training
- `utils.py` → helper functions

### 🚀 How to Run

```bash
pip install -r requirements.txt
python train_unet.py
